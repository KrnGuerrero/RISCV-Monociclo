# RISCV-Monociclo
Desarrollo de un procesador monociclo compatible con el conjunto de instrucciones RV32I de RISC-V. El énfasis estará en la creación/ajuste de módulos, su verificación con Icarus Verilog, y el análisis de formas de onda en WaveTrace (VSCode). De forma opcional, se sugiere preparar el flujo para su posterior prueba en FPGA.
